en effet ce projet permet de faire un crud simple avec le framework spring boot
on a trois entite a savoir le client les produit et la commande , lorsqu un client passe une commande la quantite en stock du produit diminue automatiquement 
et pour lancer le projet vous allez juste dans le fichier application.properties et vous changer les information de la bd qui est progres sql et vous allez dans spg admin creer une nouvelle base de donner , ce qui et bien avec spring boot c est quq il genere automatiquement les tables dans la bd
